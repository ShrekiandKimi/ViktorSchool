Алгебра

Основные понятия

Алгебра - раздел математики, который изучает операции над числами и переменными, а также их свойства.

Ключевые темы

  • Линейные уравнения и неравенства
  • Квадратные уравнения
  • Системы уравнений
  • Прогрессии (арифметическая и геометрическая)
  • Функции и их графики
  • Степени и корни

Формулы квадратного уравнения

ax² + bx + c = 0
D = b² - 4ac
x = [-b ± √D] / (2a)

Пример решения квадратного уравнения

Задача: Решить уравнение x² - 5x + 6 = 0

Решение:

1. Находим дискриминант: D = (-5)² - 4·1·6 = 25 - 24 = 1

2. Корни уравнения: x₁ = [5 + √1]/2 = 6/2 = 3; x₂ = [5 - √1]/2 = 4/2 = 2

Ответ: x₁ = 3, x₂ = 2

Арифметическая прогрессия

Арифметическая прогрессия - последовательность чисел, в которой разность между последующим и предыдущим членами постоянна.

aₙ = a₁ + d(n-1)
Sₙ = n/2 · (a₁ + aₙ)

Геометрия

1. Углы и прямые

1.1 Смежные углы

Два угла с общей стороной, где две другие стороны образуют прямую линию.

∠AOB + ∠BOC = 180°

1.2 Вертикальные углы

Углы, образованные пересекающимися прямыми, равны между собой.

∠AOB = ∠COD

1.3 Параллельные прямые

Свойства:

  • Накрест лежащие углы равны: ∠1 = ∠2
  • Соответственные углы равны: ∠1 = ∠4
  • Односторонние углы в сумме 180°: ∠1 + ∠3 = 180°

2. Треугольники

2.1 Основные свойства

  • Сумма углов: ∠A + ∠B + ∠C = 180°
  • Внешний угол равен сумме двух несмежных внутренних

2.2 Признаки равенства

  • По двум сторонам и углу между ними (SAS)
  • По стороне и двум прилежащим углам (ASA)
  • По трём сторонам (SSS)

2.3 Медиана, биссектриса, высота

  • Медиана делит треугольник на равновеликие части
  • Биссектриса делит противоположную сторону пропорционально прилежащим сторонам
  • Высота - перпендикуляр из вершины к противоположной стороне

2.4 Равнобедренный треугольник

  • Углы при основании равны
  • Биссектриса/медиана/высота к основанию совпадают

2.5 Прямоугольный треугольник

c² = a² + b² (теорема Пифагора)

Катет против 30° равен половине гипотенузы

2.6 Площадь треугольника

S = ½ · a · hₐ
S = ½ · a · b · sinα
S = √[p(p-a)(p-b)(p-c)] (формула Герона)

3. Четырёхугольники

3.1 Параллелограмм

Свойства:

  • Противоположные стороны равны и параллельны
  • Диагонали делятся точкой пересечения пополам
S = a · hₐ = a · b · sinα

3.2 Прямоугольник

Параллелограмм с прямыми углами

S = a · b

3.3 Ромб

Параллелограмм с равными сторонами

S = ½ · d₁ · d₂

3.4 Квадрат

Прямоугольник с равными сторонами

S = a²
d = a√2

3.5 Трапеция

Четырёхугольник с одной парой параллельных сторон

S = ½ · (a + b) · h

4. Окружность

4.1 Основные понятия

  • Центральный угол равен дуге, на которую опирается
  • Вписанный угол равен половине дуги, на которую опирается

4.2 Касательные и хорды

  • Касательная перпендикулярна радиусу в точке касания
  • Отрезки касательных из одной точки равны

4.3 Вписанная и описанная окружности

  • Центр вписанной окружности - точка пересечения биссектрис
  • Центр описанной окружности - точка пересечения серединных перпендикуляров

4.4 Площади круга и сектора

Sкруга = πR²
Sсектора = (πR² · α)/360°

5. Важные теоремы

5.1 Теорема Фалеса

Параллельные прямые отсекают на сторонах угла пропорциональные отрезки

AB/BC = A₁B₁/B₁C₁

5.2 Теорема синусов

a/sinα = b/sinβ = c/sinγ = 2R

5.3 Теорема косинусов

a² = b² + c² - 2bc·cosα

Пример решения геометрической задачи

Задача: В треугольнике ABC угол C = 90°, AB = 10 см, BC = 6 см. Найти AC.

Решение:

По теореме Пифагора:

AC² = AB² - BC² = 10² - 6² = 100 - 36 = 64

AC = √64 = 8 см

Ответ: 8 см

Теория вероятностей

Основные понятия

Теория вероятностей - раздел математики, изучающий случайные события и их закономерности.

Ключевые темы

  • Классическое определение вероятности
  • Комбинаторика (перестановки, сочетания, размещения)
  • Геометрическая вероятность
  • Сложение и умножение вероятностей
  • Условная вероятность
  • Случайные величины

Формула классической вероятности

P(A) = m/n

Где m - число благоприятных исходов, n - общее число равновозможных исходов.

Пример решения задачи на вероятность

Задача: В коробке 5 синих и 3 красных шара. Какова вероятность вытащить синий шар?

Решение:

Общее количество шаров: 5 + 3 = 8

Число благоприятных исходов (синих шаров): 5

P(синий) = 5/8 = 0.625

Ответ: 0.625 или 62.5%

Формулы комбинаторики

Перестановки: Pₙ = n!
Размещения: Aₙᵏ = n!/(n-k)!
Сочетания: Cₙᵏ = n!/(k!(n-k)!)
